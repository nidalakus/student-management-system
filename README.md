# Student Management System

A Java OOP project for managing student records.

## Classes
- **Person** - Base class with name and age
- **Student** - Extends Person, adds student ID and GPA
- **StudentManager** - Manages student list with ArrayList
- **Main** - Interactive menu-driven interface

## Features
- Add and remove students
- Search student by name
- List all students
- GPA tracking

## Language
Java
